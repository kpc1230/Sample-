Capture for JIRA test session notes allow special "#tags" to be inserted so you give more meaning to your notes and find them more easily later.

Capture for JIRA ships with 4 predefined tags.

  • #? which represents a Question
  • #f which represents a Follow Up
  • #! which represents an Assumption
  • #i which represents an Idea

You can define your own tags however.

A special tag is simply a word that begins with the # character. For #example this sentence #contains three #tags.

Notes can also contain Atlassian wiki markup. For example you can use *strong*, _emphasis_ and ^superscript^.

You can also use macros such as the {quote} and {noformat}.
